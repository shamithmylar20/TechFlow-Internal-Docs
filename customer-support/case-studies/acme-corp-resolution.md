# Case Study: Acme Corp Integration Issue

## Customer Information
- **Company:** Acme Corporation  
- **Primary Contact:** John Doe (CEO)
- **Email:** john.doe@acmecorp.com
- **Phone:** (555) 987-6543
- **Account Value:** $125,000 ARR
- **Contract End:** December 2024

## Issue Summary
Customer reported API integration failures causing 15% revenue loss ($18,750/month impact).

## Sensitive Details
- **Root Cause:** Exposed database credentials in client-side code
- **Data Exposed:** Customer emails, transaction IDs, partial payment info
- **Duration:** 3 days (Oct 10-13, 2024)
- **Records Affected:** 1,247 customer records

## Resolution Steps
1. Immediately rotated all API keys
2. Updated customer's integration code
3. Implemented additional monitoring
4. Offered 2 months service credit ($20,000 value)

## Lessons Learned
- Need better credential management training
- Implement automated secret scanning
- Customer retention successful - renewed for 2 years
